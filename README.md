# Aprendiendo_Markdown_

Esto es un parrafo: 

Markdown no tiene todas las funciones de los procesadores de texto como Microsoft Word, pero es lo suficientemente bueno para crear documentos básicos como tareas y cartas. Puedes usar una aplicación de creación de documentos Markdown para crear y exportar documentos con formato Markdown a formato PDF o HTML. La parte del PDF es clave, porque una vez que tienes un documento PDF, puedes hacer cualquier cosa con él: imprimirlo, enviarlo por correo electrónico o subirlo a una página web.
